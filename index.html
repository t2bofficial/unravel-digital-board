🔌 Enable Live Collaboration

Connect Supabase so all 17+ team members see card moves, updates, and new cards in real time — no refresh needed. Free forever at your scale.

1
Create a free Supabase project
Go to supabase.com → New Project. The free tier supports up to 50,000 monthly active users.
2
Run the schema SQL
In Supabase → SQL Editor, paste the contents of supabase-schema.sql (in your download) and hit Run.
3
Add your keys to index.html
// Near top of <script> in index.html:
const SUPABASE_URL = 'https://xxxx.supabase.co';
const SUPABASE_KEY = 'eyJhbGci...';
Find these at: Supabase Dashboard → Settings → API
4
Invite your 17+ teammates
Share the Vercel URL. Add teammates under Supabase → Authentication → Users. Supports magic link (email) or Google OAuth — no passwords required.
💰 Cost estimate for 17–25 users
Database usage: ~7 MB/year (Supabase free = 500 MB). Bandwidth: well under 5 GB/month free limit. Running cost: ₱0/month. Only upgrade to Pro (~$25/mo) if you add file attachment storage or need daily backups.
